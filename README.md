# Windows Programming

## 과목 소개

기본적인 C# 문법과 객체지향 프로그래밍(OOP)을 익힌 후, Windows Forms를 이용해 윈도우 환경에서 실행되는 응용 소프트웨어 개발 방법을 학습합니다.

## 학습 목표

1. C# 언어가 가지는 고유한 특징 및 .NET 개발환경을 이해할 수 있습니다.

2. 윈도우 폼을 이용해 GUI를 설계하고 구현하는 실습을 진행합니다.

3. 배운 내용을 바탕으로 실제 윈도우 애플리케이션을 설계하고 개발하는 능력을 기릅니다.

## 주요 학습 내용

- C# 기본 문법
- 클래스와 객체
- 상속, 다형성, 캡슐화 등 객체지향 프로그래밍
- 이벤트 처리
- Windows Forms GUI 설계
- 버튼, 텍스트박스, 리스트박스 등 컨트롤 사용
- 간단한 응용 프로그램 제작 프로젝트

## 개발 환경

- Language : C#
- Framework : .NET
- IDE : Visual Studio
- GUI : Windows Forms
