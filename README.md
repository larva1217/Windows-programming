# Windows Programming

## 과목 소개

기본적인 C# 문법과 객체지향 프로그래밍(OOP)을 익힌 후, Windows Forms를 이용해 윈도우 환경에서 실행되는 응용 소프트웨어 개발 방법을 학습합니다.

## 교과 목표

1. C# 언어의 기초를 이해하고, 기본 문법 및 객체지향 프로그래밍(OOP)을 활용하여 효율적인 코드 작성이 가능합니다.

2. Windows Forms를 이용한 GUI 애플리케이션 개발 능력을 함양합니다.

3. 이벤트 기반 프로그래밍을 이해하고, UI 컨트롤(버튼, 텍스트박스, 리스트박스 등)을 활용하여 사용자 친화적인 인터페이스를 설계하고 구현할 수 있습니다.

4. 파일 입출력 및 멀티스레드 기법을 이해하고, 실제 프로그램에 적용할 수 있습니다.

5. 실전 프로젝트를 통해 실제 애플리케이션을 설계하고 구현하는 경험을 제공하여 문제 해결 능력과 팀워크를 배양합니다.

## 주요 학습 내용

- C# 기본 문법
- 클래스와 객체
- 상속, 다형성, 캡슐화 등 객체지향 프로그래밍
- 이벤트 처리
- Windows Forms GUI 설계
- 버튼, 텍스트박스, 리스트박스 등 UI 컨트롤 활용
- 파일 입출력
- 멀티스레드 프로그래밍
- 실전 프로젝트 기반 응용 프로그램 제작

## 개발 환경

- Language : C#
- Framework : .NET
- IDE : Visual Studio
- GUI : Windows Forms
